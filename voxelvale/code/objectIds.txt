Steps for adding 'new' blocks:
1) If it's simple enough add to "blockwall.js".
2) Set objectNumber to the next available number.
3) Add to the list above.
4) Add to 'simpleBlocks' array in "blockwall.js".
5) Add to 'BLOCK_OBJNUMS' in "saveload.js".

Steps for adding 'old' blocks:
1) Add class.
2) Set objectNumber to the next available number.
3) Add to the list above.
4) Add to 'BLOCK_OBJNUMS' in "saveload.js".
5) Add in 'send_block()' in main.js.
(If not accounted for in send_data() do the following two steps)
6) Save starting vertices as 'NAMEOFBLOCK_START'.
7) Change index in block class to 'NAMEOFBLOCK_START'.

Steps for adding recipes:
1) Add class to "recipes.js".
2) Set objectNumber to the next available number.
3) Add to the list above.
4) Add to 'RECIPE_OBJNUMS' in "recipes.js".

Steps for adding non-actionable items:
1) Add class to "non-actionable.js"
2) Set objectNumber to the next available number.
3) Add to the list above.
4) Add to 'nonActionableItems' array in "nonactionableitems.js".
5) Add to 'ITEM_OBJNUMS' in "nonactionableitems.js".
6) Design build function in static method 'sendData()'.

Steps for adding tools:
1) Add class
2) Set objectNumber to the next available number.
3) Add to the list above.
4) Add to 'ITEM_OBJNUMS' in "nonactionableitems.js".
5) Design build function.
